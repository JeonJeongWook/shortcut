윈도우 10 단축키 -----------------------------------

윈도우 키 + 숫자(1~9)           작업표시즐에 있는 프로그램 실행(여러개가 실행 되어있는 경우 윈도우 키 누른 상태에서 1 여러번 클릭)
윈도우 키 + Shift + 숫자(1~9)   프로그램 새창으로 실행
윈도우 키 + Alt   + 숫자(1~9)   작업표시줄에서 우클릭 한거랑 같음

윈도우 키 + 방향키              창 맞추기
윈도우 키 + shift + (좌/우)     창 이동(듀얼 모니터일 경우)
윈도우 키 + shift + 상		창 폭은 그대로 두고 높이만 최대화

윈도우 키 + L                   화면 잠금
윈도우 키 + D                   바탕화면 보기(다시 누르면 내리기 전 화면으로 켜짐)
윈도우 키 + [HOME]키		현재 활성화 창 제외하고 전부 최소화

윈도우 키 + V			클립보드 보기
윈도우 키 + R                   실행창(regedit, magnify, control, calc, notepad)
윈도우 키 + E                   파일 탐색기 실행

윈도우 키 + ctrl + D            가상 데스크 톱 만들기
윈도우 키 + ctrl + [← / →]      가상 데스크 톱 전환
윈도우 키 + ctrl + [F4]         가상 데스크 톱 지우기


크롬 브라우저 단축키 --------------------------------
Ctrl + H    방문 기록
Ctrl + D    북마크
Ctrl + N    새 창
Ctrl + T    새 탭
Ctrl + W    현재 탭 닫기
Ctrl + L    주소창 가기
Ctrl + J    다운로드 탭 열기
Ctrl + U    홈페이지 소스 보기

Ctrl + [1~8]                해당 숫자 탭 전환
Ctrl + 9                    마지막 탭 전환
Ctrl + [Page Up/Down]키     탭 전환
Ctrl + [Tab]키              탭 전환(shift키랑 같이 누를 경우 역방향)

Ctrl + [+]키    확대(Ctrl + 휠 위로)
Ctrl + [-]키    축소(Ctrl + 휠 아래로)
Ctrl + 숫자 0   화면크기 100%

Ctrl + Shift + B    북마크 바 On/Off
Ctrl + Shift + N    시크릿 모드
Ctrl + Shift + T    최근에 닫은 탭 열기

Alt + [Home]키  시작 페이지로 이동
Alt + [← / →]   이전페이지 / 다음페이지 이동

Ctrl  + 좌클릭	새 탭으로 열기
Shift + 좌클릭 	새 창으로 열기
Ctrl  + Shift + C	요소보기
